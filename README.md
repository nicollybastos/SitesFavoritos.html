**Meus Sites Favoritos 🌐**

Este é um projeto de página web pessoal desenvolvido para listar e organizar links de acesso rápido aos sites e redes sociais mais utilizados. A interface foi projetada com um layout minimalista, focado na experiência do usuário e na organização visual em coluna.

🚀 Tecnologias Utilizadas
HTML5: Estruturação semântica do conteúdo.

CSS3: Estilização personalizada, incluindo:

Flexbox: Utilizado para garantir o alinhamento vertical centralizado de todos os elementos.

SVG dinâmico: Implementação de fundo decorativo no cabeçalho.

Design Responsivo: Uso de fontes e containers que se adaptam à visualização.

🎨 Características do Projeto
Cabeçalho Personalizado: Inclui um fundo rosa suave com ondas estilizadas e uma barra de identificação roxa para o nome da usuária: NICOLLY GÓES BASTOS.

Layout em Coluna: Os sites favoritos (LinkedIn, TikTok, Instagram, WhatsApp e Spotify) são exibidos de forma vertical para facilitar a navegação em dispositivos móveis e desktop.

Identidade Visual: Uso de divisores em magenta e botões arredondados com contraste visual para facilitar o clique.

Fidelidade Visual: Logos com fundo transparente e ajuste de escala proporcional (object-fit: contain).
